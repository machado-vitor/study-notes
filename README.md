# study-notes
The purpose of this repo is to organize my personal studies.
Each topics has notes, cheatsheets, exercises, and small projects.

## Structure

| Folder | Description |
|---------|--------------|
| [`notes/`](notes) | Detailed explanations and summaries for each topic. |
| [`cheatsheets/`](cheatsheets) | Quick references and syntax summaries. |
| [`books/`](books) | Notes and takeaways from technical books. |
| [`exercises/`](exercises) | Algorithm problems, and coding challenges. |
| [`projects/`](projects) | Small demos or experiments to apply what I learn. |
| [`snippets/`](snippets) | Small, reusable code examples. |
| [`journal/`](journal) | Daily/weekly logs of what I studied and reflections. |
| [`resources/`](resources) | Curated lists of articles, videos, and papers. |
